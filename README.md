# Pharmacy Instructions App

A Streamlit-based web application that provides clear, multilingual medicine instructions with reminders and visual/audio support.

## Features
- Prescription upload or manual entry
- Medicine dosage instructions
- Multilingual support (English, Kannada, Hindi)
- Reminder alerts
- Visual icons and audio guidance

## Technologies Used
 This demo uses synthetic data. In production, we integrate Google Cloud Vision API for prescription scanning, Firestore for medicine data, and Translation API for real-time language support. Python for Coding.



## How to Run
streamlit run app.py
